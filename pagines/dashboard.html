Documentació de la pàgina: DashboardPage

Descripció general

Aquesta pàgina mostra les dades personals de l'usuari i les lligues en les quals està inscrit. Permet consultar informació detallada de l'usuari, com el nom, els cognoms, l'email, l'edat, el número de federat i si té privilegis d'administrador. A més, l'usuari pot veure les lligues a les quals està inscrit i obtenir detalls sobre les partides i la classificació dels jugadors per cada lliga.

Estructura del component

Propietats (Data)

  • userData: Objecte que conté les dades de l'usuari. Inclou les següents subpropietats:
    • userData.nom: Nom de l'usuari (string).
    • userData.cognoms: Cognoms de l'usuari (string).
    • userData.email: Correu electrònic de l'usuari (string).
    • userData.edat: Edat de l'usuari (número).
    • userData.num_federat: Número de federat de l'usuari (número).
    • userData.admin: Valor booleà que indica si l'usuari és administrador (boolean).
    • userData.lligues: Llistat de les lligues on l'usuari està inscrit (Array).
  • partides: Llistat de les partides d'una lliga específica (Array).
  • tipusTorneig: Tipus de torneig seleccionat (Lliga o TorneigEliminatori) (string).
  • jugadors: Llistat de jugadors amb la classificació d'una lliga (Array).

Metodes

getUserData

Descripció: Aquest mètode obté les dades de l'usuari des d'una API. Utilitza l'ID de l'usuari emmagatzemat en el localStorage per realitzar la sol·licitud a l'API i omple l'objecte userData amb la resposta.

onLligaClick

Descripció: Aquest mètode gestiona el clic en una lliga per obtenir les partides associades a aquesta lliga i actualitzar la classificació dels jugadors. Quan l'usuari selecciona una lliga, el sistema realitza una sol·licitud a l'API per obtenir les dades relacionades amb la lliga seleccionada.

Estils (CSS)

Els estils d'aquesta pàgina utilitzen classes d'utilitat de Tailwind CSS per gestionar l'aspecte visual. S'han definit algunes classes personalitzades per gestionar l'alineació del text, com .text-left i .text-center.
