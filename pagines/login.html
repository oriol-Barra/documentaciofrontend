Documentació de la pàgina: IniciarSessio

Descripció general

Aquesta pàgina permet a l'usuari iniciar sessió en el sistema mitjançant les seves credencials (email i contrasenya). En cas que les credencials siguin correctes, el sistema guarda les dades de l'usuari, inclòs el token d'autenticació, i redirigeix l'usuari a la pàgina del seu tauler de control (/dashboardPage).

Estructura del component

Propietats (Data)

  • email: Correu electrònic de l'usuari (string).
  • password: Contrasenya de l'usuari (string).

Metodes

submit

Descripció: Aquest mètode s'encarrega de gestionar el procés d'inici de sessió de l'usuari. Realitza una sol·licitud POST al servidor amb les credencials de l'usuari (email i contrasenya) i, si la resposta és positiva (status 200), guarda la informació de l'usuari al localStorage (incloent el token d'autenticació, ID de l'usuari, nom de l'usuari i l'estat d'administrador). Després, neteja els camps del formulari i redirigeix a la pàgina de l'usuari (/dashboardPage). En cas d'error, es mostra un missatge a l'usuari.

Estils (CSS)

Aquesta pàgina utilitza les classes d'utilitat de Tailwind CSS per donar estil als elements visuals. Les classes utilitzades inclouen .bg-white, .p-8, .rounded-lg, .shadow-lg, entre altres per a donar forma i estil a la pàgina.
